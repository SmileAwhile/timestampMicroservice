Timestamp Microservice

Instructions:

Pass a Javascript Date object or unix timepstamp in the url and get the unix timestamp and date object for that time

Example:

https://tsmic.herokuapp.com/December%2015,%202015

https://tsmic.herokuapp.com/1450166400

return example:

{"unixtime":1450166400,"natural":"December 15, 2015"}
